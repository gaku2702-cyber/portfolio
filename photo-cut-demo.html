体験版について
  • このページは認識プレビューです。検出の考え方（黒紙と写真の明度差）は Python 版と同じですが、解像度・検出精度は Python 版と異なります（プレビュー用に画像を縮小して解析しています）。
  • 赤枠は矩形のおおよその位置です。斜めに置いた写真は角に黒が残る場合があります。
  • Python 版（本番アプリ）では、切り出しに加えて傾き・歪み補正（台形補正）を行い、写真を正位置に整えてから保存します。1200dpi での書き出しにも対応しています。
