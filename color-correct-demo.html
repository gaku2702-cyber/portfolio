1. 写真を読み込む

体験デモについて
  • ブラウザ上で解析プレビューと補正イメージを確認できます。
  • 本番の Windows アプリでは、写真をドロップ → Photoshop 起動 → 画像を開く → 調整レイヤーで色補正 → PSD（レイヤー付き）保存まで自動化します。
  • このデモはプレビュー用です。Photoshop 連携や PSD 書き出しは Windows 版で行います。
