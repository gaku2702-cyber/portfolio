# お問い合わせフォーム（Google Apps Script）

送信先: **smartdtp.studio.works@gmail.com**

## 初回セットアップ（必須）

GAS エディタで **この順番** に実行してください。

### 1. コードを貼り付け

`contact-form.gs` の内容を GAS プロジェクトにすべてコピー。

### 2. スプレッドシート受信箱を作成

関数 **`setupContactSheet`** を選んで **実行**。

- 初回は「権限の承認」を求められたら許可
- 実行ログにスプレッドシート URL が表示される
- 以降、フォーム送信は **必ずこのシートにも記録** される（メール不達の保険）

### 3. メール送信テスト

関数 **`testSendEmail`** を実行。

- `smartdtp.studio.works@gmail.com` にテストメールが届くか確認
- 届かない場合 → 迷惑メール・プロモーションタブも確認
- GAS の **実行数** タブでエラー内容を確認

### 4. ウェブアプリとしてデプロイ

- **デプロイ → 新しいデプロイ → ウェブアプリ**
- 実行ユーザー: **自分**
- アクセス: **全員**
- URL を `index.html` の `GAS_URL` に設定して push

---

## メールが届かないとき

| 確認項目 | 対処 |
|---|---|
| 送信成功なのに届かない | スプレッドシート受信箱に行が増えているか確認 |
| シートにも増えない | GAS 未デプロイ or `GAS_URL` が古い |
| シートには増えるがメールだけ来ない | `testSendEmail` を再実行。Gmail の送信上限（1日100通） |
| 権限エラー | GAS で再承認 → **新バージョン** で再デプロイ |
| 自分宛て送信 | デプロイアカウント＝受信アカウントの場合、**送信済み** に入ることも |

## サイト側

`index.html` は iframe + form POST で送信し、GAS から `postMessage` で成功/失敗を受け取ります。以前の「常に成功表示」は修正済みです。
